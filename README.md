# API Portfólio

## Como desenvolver?

1. Clone o repositório
2. Acesse a pasta
3. Instale as dependências
4. Instale os hooks
5. Rode o projeto 

```bash
git clone git@github.com:jhonifreitas/portfolio-api.git
cd portfolio-api
yarn i
npm run server
```
